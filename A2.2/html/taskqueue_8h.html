Parameters

tDie Task, die in die Queue eingefügt wird.
prioDie Priorität von t. Es werden die Prioritäten von Message Queues unter LINUX verwendet.
blockingDieser Parameter legt fest, ob der Funktionsaufruf im Fall einer vollen Task Queue blockiert.
Returns
Der return ist 0, wenn die Task eingefügt werdeYn konnte. Ist die Queue voll und blocking == true, dann wird die Task nicht eingefügt und eine 1 zurück gegeben. Im Fehlerfall wird -1 zurück gegen und errno enthält eine Beschreibung des Fehlers gemäss der Funktion mq_send.
